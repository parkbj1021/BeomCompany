# beom-plan 지식 저장소 v1

## 개요

플래닝/아키텍처 학습 경험을 기록합니다.
학습이 누적되면 VERSION 파일을 증가시키고 새 도메인 버전(beom-plan-v2)을 만듭니다.

## 학습 카테고리

- **PDCA 사이클**: Plan → Design → Do → Check → Act
- **Opus 플래닝**: 깊은 분석과 구조화된 계획 생성
- **Agent Teams 설계**: 팀 구성, 역할 분리, 병렬 실행
- **스킬/플러그인 설계**: 재사용 가능한 Claude Code 스킬 패턴
